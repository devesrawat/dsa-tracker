2. TWO POINTERS ðŸ‘‰ðŸ‘ˆ

When to Use

Intuition

Exploit sorted order to converge pointers from ends or fast/slow to avoid nested loops, reducing time by partitioning space efficiently.

Core Templates

Collision (Sorted Sum)

def pair_sum(nums, target):
    nums.sort() # If unsorted
    left, right = 0, len(nums) - 1
    while left < right:
        curr_sum = nums[left] + nums[right]
        if curr_sum == target:
            return [left, right]
        elif curr_sum < target:
            left += 1 # Need larger
        else:
            right -= 1 # Need smaller
    return []

Fast-Slow (Remove Dups)

def remove_dups(nums):
    if not nums: return 0
    slow = 1 # Assume sorted
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:
            nums[slow] = nums[fast]
            slow += 1
    return slow # New length

3. SLIDING WINDOW ðŸªŸ

When to Use

  • Fixed/variable subarrays (max sum, longest valid)
  • Substring matches/conditions
  • Anagrams/permutations in stream
  • Signal: "longest/max in subarray" or "at most K"

Intuition

Maintain a dynamic window that expands/contracts to satisfy conditions, amortizing adjustments for O(n) traversal over all possible subarrays.

Core Templates

Fixed Size

def max_subarray_k(arr, k):
    if len(arr) < k: return 0
    window = sum(arr[:k])
    maxv = window
    for i in range(k, len(arr)):
        window += arr[i] - arr[i - k] # Slide: add right, remove left
        maxv = max(maxv, window)
    return maxv

Dynamic (Condition)

from collections import Counter
def longest_valid(s, k): # e.g., at most K distinct
    left = 0
    count = Counter()
    max_len = 0
    for right in range(len(s)):
        count[s[right]] += 1
        while len(count) > k: # Invalid: shrink
            count[s[left]] -= 1
            if count[s[left]] == 0: del count[s[left]]
            left += 1
        max_len = max(max_len, right - left + 1)
    return max_len

Traps

  • âŒ Nested loops â†’ O(nÂ²); always one pass
  • âŒ Shrink only when invalid; expand always
  • âŒ Empty: if not s: return 0

def next_greater(nums):
    stack = [] # Decreasing monotonic
    res = [-1] * len(nums)
    for i in range(len(nums) - 1, -1, -1): # Right to left
        while stack and stack[-1] <= nums[i]:
            stack.pop()
        res[i] = stack[-1] if stack else -1
        stack.append(nums[i])
    return res

5. BINARY SEARCH ðŸ”

When to Use

  • Sorted arrays for exact/range search
  • Find boundaries/peaks
  • Optimize parameters (min speed, etc.)

Intuition

Halve search space repeatedly on monotonic functions, turning exponential brute-force into logarithmic by discarding impossible halves.

Core Templates

Standard Search

def search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2 # No overflow
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

Lower Bound

def lower_bound(arr, target):
    left, right = 0, len(arr)
    while left < right: # < for insertion point
        mid = left + (right - left) // 2
        if arr[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left # First >= target

Answer Optimization

def min_eating_speed(piles, h):
    left, right = 1, max(piles)
    while left < right:
        mid = left + (right - left) // 2
        if sum(math.ceil(pile / mid) for pile in piles) <= h:
            right = mid # Try smaller
        else:
            left = mid + 1
    return left

Traps

  • âŒ left < right misses single element; use <=
  • âŒ Overflow: (l + r) // 2 not l + r >> 1 in Python
  • âŒ Infinite loop: Always mid Â±1, not mid

def merge(intervals):
    if not intervals: return []
    intervals.sort(key=lambda x: x[0]) # By start
    merged = [intervals[0][:]] # Copy

        last = merged[-1]
        if curr[0] <= last[1]: # Overlap/contain

        else:
            merged.append(curr[:])

def min_rooms(intervals):
    starts = sorted([i[0] for i in intervals])
    ends = sorted([i[1] for i in intervals])
    rooms, e_i = 0, 0
    for s in starts:
        if s >= ends[e_i]:
            e_i += 1 # Free room
        else:
            rooms += 1
    return rooms

def k_largest(nums, k):
    heap = nums[:k] # Initial min-heap of k

        if num > heap[0]:
            heapq.heapreplace(heap, num) # Pop min, push larger
    return heap # Extract all or heap[0] for kth

class MedianFinder:
    def __init__(self):
        self.small = [] # Max-heap (negated)
        self.large = [] # Min-heap
    def add_num(self, num):
        heapq.heappush(self.small, -num)
        heapq.heappush(self.large, -heapq.heappop(self.small))
        if len(self.large) > len(self.small):
            heapq.heappush(self.small, -heapq.heappop(self.large))

Core Template

Subsets Backtrack

def subsets(nums):
    res = []
    def backtrack(start, path):
        res.append(path[:]) # Copy!
        for i in range(start, len(nums)):
            path.append(nums[i])
            backtrack(i + 1, path) # No reuse
            path.pop() # Backtrack
    backtrack(0, [])
    return res

def permute(nums):
    res = []
    used = [False] * len(nums)
    def backtrack(path):
        if len(path) == len(nums):
            res.append(path[:])
            return
        for i in range(len(nums)):
            if used[i]: continue
            used[i] = True
            path.append(nums[i])
            backtrack(path)
            path.pop()
            used[i] = False
    backtrack([])
    return res

Traps

def topo_sort(graph, n):
    indeg = [0] * n
    for u in graph:
        for v in graph[u]:
            indeg[v] += 1
    q = deque([i for i in range(n) if indeg[i] == 0])
    order = []
    while q:
        u = q.popleft()
        order.append(u)
        for v in graph.get(u, []):
            indeg[v] -= 1
            if indeg[v] == 0:
                q.append(v)
    return order if len(order) == n else [] # Cycle if not

Traps

  • âŒ Cycles: Topo fails if len(order) < n
  • âŒ Disconnected: Start DFS/BFS from all nodes

def longest_common_subsequence(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

5 Steps

  1. State: dp[i][j] = max profit up to i
  2. Recur: dp[i] = max(dp[i-1], dp[i-2] + val)
  3. Base: dp[0]=0, dp[1]=val[0]
  4. Order: For i in 1 to n (dependencies first)
  5. Space: 2Dâ†’1D if row-only deps

Traps

  • âŒ Base cases wrong (off-by-one in init)
  • âŒ Iteration order: Compute small to large
  • âŒ Bounds: Use n+1 sizing
  • âŒ Space: Optimize to O(min(m,n)) when possible

Merge Sort Count

def merge_sort(arr):
    if len(arr) <= 1: return arr, 0 # Inversions
    mid = len(arr) // 2
    left, inv_l = merge_sort(arr[:mid])
    right, inv_r = merge_sort(arr[mid:])
    merged, inv_m = merge(left, right)
    return merged, inv_l + inv_r + inv_m
def merge(left, right):
    merged = []
    inv = i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            merged.append(left[i])
            i += 1
        else:
            merged.append(right[j])
            inv += len(left) - i # All remaining left > right[j]
            j += 1
    return merged + left[i:] + right[j:], inv

Sorted Array to BST

def sorted_array_to_bst(nums):
    def helper(left, right):
        if left > right: return None
        mid = (left + right) // 2
        root = TreeNode(nums[mid])
        root.left = helper(left, mid - 1)
        root.right = helper(mid + 1, right)
        return root
    return helper(0, len(nums) - 1)

Traps

  • âŒ Recursion depth: O(log n) ok, but stack overflow on n=10^5
  • âŒ Merge efficiency: O(n) per level, total O(n log n)
  • âŒ Off-mid: Balance tree with mid choice
  • âŒ Base: left > right not ==
